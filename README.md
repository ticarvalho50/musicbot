PT/BR

Para roda-lo precisamos criar um bot de discord, para isso siga os passos:
Passo 1: Crie uma conta no Discord

Se você ainda não tiver uma conta no Discord, crie uma em https://discord.com/register.

Passo 2: Crie um servidor (opcional)

Se você ainda não tiver um servidor no Discord onde deseja que seu bot seja usado, crie um novo servidor. Para isso, clique no ícone "+" na barra lateral do Discord e escolha a opção "Criar um servidor".

Passo 3: Acesse o Discord Developer Portal

Acesse o Discord Developer Portal em https://discord.com/developers/applications. Faça login com sua conta Discord, se necessário.

Passo 4: Crie uma nova aplicação

Clique em "New Application" (Nova Aplicação) e dê um nome à sua aplicação. Este nome será o nome do seu bot.

Passo 5: Configure o bot

No painel da sua aplicação, vá para a guia "Bot" e clique em "Add Bot" (Adicionar Bot). Isso transformará sua aplicação em um bot Discord.

Passo 6: Copie o Token do Bot

Na seção "Token", clique em "Copy" (Copiar) para copiar o token do seu bot. Guarde-o em um local seguro, pois você precisará dele para a configuração do bot.

Passo 7: Convide o bot para o seu servidor

Na guia "OAuth2" do seu bot, marque as permissões que deseja que seu bot tenha e copie o URL gerado. Acesse este URL em seu navegador e siga as instruções para convidar o bot para o seu servidor.

Passo 8: Configure seu ambiente de desenvolvimento

Para programar seu bot, você precisará de um ambiente de desenvolvimento. Você pode escolher uma linguagem de programação, como JavaScript (Node.js), Python, Java, etc. Instale as bibliotecas necessárias para se conectar à API do Discord e trabalhar com bots.

Passo 9: Codifique o bot

Escreva o código do seu bot para definir como ele se comportará no servidor. Você precisará se familiarizar com a API do Discord para realizar tarefas como responder a comandos, interagir com membros e gerenciar canais.

Passo 10: Hospede o bot

Hospede seu bot em um servidor ou plataforma de hospedagem. Certifique-se de que o bot esteja sempre online para responder às solicitações.

Passo 11: Execute o bot

Inicie o bot e conecte-o ao servidor do Discord usando o token que você copiou anteriormente.

Passo 12: Teste e ajuste

Teste seu bot no servidor Discord e faça ajustes conforme necessário para garantir que ele funcione conforme o esperado.


##ENG##

Step 1: Create a Discord Account

If you don't already have a Discord account, create one at https://discord.com/register.

Step 2: Create a Server (Optional)

If you don't have a Discord server where you want your bot to be used, create a new server. Click the "+" icon in the Discord sidebar and choose "Create a Server."

Step 3: Access the Discord Developer Portal

Go to the Discord Developer Portal at https://discord.com/developers/applications. Log in with your Discord account if prompted.

Step 4: Create a New Application

Click "New Application" and give it a name. This name will be your bot's name.

Step 5: Set Up the Bot

In your application's dashboard, go to the "Bot" tab and click "Add Bot." This will turn your application into a Discord bot.

Step 6: Copy the Bot Token

In the "Token" section, click "Copy" to copy your bot's token. Keep it in a safe place as you'll need it for bot configuration.

Step 7: Invite the Bot to Your Server

Under the "OAuth2" tab of your bot, select the permissions your bot needs and copy the generated URL. Paste this URL in your browser and follow the instructions to invite the bot to your server.

Step 8: Set Up Your Development Environment

To program your bot, you'll need a development environment. Choose a programming language such as JavaScript (Node.js), Python, Java, etc. Install the necessary libraries to interact with the Discord API and work with bots.

Step 9: Code the Bot

Write the code for your bot to define how it will behave on the server. You'll need to become familiar with the Discord API to perform tasks like responding to commands, interacting with members, and managing channels.

Step 10: Host the Bot

Host your bot on a server or hosting platform. Ensure that the bot is always online to respond to requests.

Step 11: Run the Bot

Start the bot and connect it to the Discord server using the token you copied earlier.

Step 12: Test and Adjust

Test your bot on the Discord server and make adjustments as needed to ensure it functions as expected.


Here are the basic commands and how to use them:

!play [search_query]: This command plays a song from YouTube. You can use it like this:

diff

    !play despacito

!showqueue: This command displays the current queue of songs.

diff

    !showqueue

!pause: Pause the currently playing song.

css

    !pause

!resume: Resume a paused song.

diff

    !resume

!clearqueue: Clear the entire song queue.

diff

    !clearqueue

!skip: Skip the currently playing song and play the next song in the queue.

diff

    !skip

!para: Stop playback entirely.

diff

    !para
